# mobiletouch

Supported event names:
  *  swipeStart
  *  swipeProgress
  *  swipe
  *  swipeLeft
  *  swipeRight
  *  swipeUp
  *  swipeDown
  *  swipeCancel
  *  tap

Tested on
  * Android 2.3.5 (ZTE Q101T)
  * Android 4.0.3 (Coolpad 8076D)
  * Android 4.2.2 (Sony S36h)
  * Windows Phone 8.1 Update (Nokia Lumia 530)
  * iOS 8.3 (iPhone 6 plus)
  * Chrome 42 (Windows 8.1)
  * IE 9 (Windows 7)
  * IE 10 (Windows 7)
  * IE 11 (Windows 8.1)
  * Firefox 37 (Windows 8.1)
